# Docs Index

เอกสารในโปรเจกต์นี้ถูกจัดใหม่ให้เหลือเฉพาะชุดที่ใช้งานจริงและตรงกับโค้ดปัจจุบัน

## เอกสารหลัก

- `ARCHITECTURE.md` ใช้อ่านโครงสร้างระบบ, design pattern, flow คำนวณ, data model และข้อจำกัดของ implementation ปัจจุบัน
- `API_TESTING_GUIDE.md` ใช้เทส API และ Postman pipeline สำหรับพิสูจน์โจทย์เรื่อง promotion stacking, extensibility และ correctness

## เอกสารฐานข้อมูล

- `../database/README.md` อธิบาย schema หลัก, index สำคัญ และ query pattern ที่ระบบใช้จริง

## ไฟล์อ้างอิงอื่นในโฟลเดอร์นี้

- `.pdf`, `.png`, `.sql`, `.txt` เป็นเอกสารอ้างอิงเดิมจากช่วงออกแบบ
- ใช้เป็น supporting material ได้ แต่ไม่ถือเป็น source of truth หลัก
- source of truth ฝั่งเอกสาร markdown ตอนนี้คือ `README.md`, `Docs/*.md`, และ `database/README.md`
